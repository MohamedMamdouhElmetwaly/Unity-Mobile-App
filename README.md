# Mobile App - Unity Project

This is a Unity project for a mobile application featuring a preset-based camera system and virtual tour capabilities.

## Features
- **Preset System**: Save and reuse camera settings and UI configurations.
- **Camera Controller**: Supports Top View and Virtual Tour modes with smooth transitions.
- **Mobile Optimized**: Includes Virtual Joystick, Touch Indicators, and Pinch-to-Zoom.
- **Project Template Utility**: Tools to quickly apply settings and verify package dependencies.

## Setup Instructions
1. Open the project in Unity 6000.4.6f1 or later.
2. Ensure the **Android** or **iOS** platform is selected in Build Settings.
3. Use the **Project Template Utility** under `Tools > Project Template > Setup Utility` to finalize project settings.
